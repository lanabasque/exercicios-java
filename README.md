## ▶️ Como executar

Com o JDK instalado, dentro da pasta do repositório:

```bash
javac NomeDoArquivo.java
java NomeDoArquivo
```

Troque `NomeDoArquivo` pelo nome do exercício que quiser rodar (sem o `.java`).
